# 문제

사람들의 공부 기록이 담긴 2차원 문자열 배열 `arr`가 주어집니다.

`arr[i][0]`에는 i번째 사람의 **이름**이 주어지고, `arr[i][1]`부터 `arr[i][m]`까지는 각 날의 공부 결과를 나타내는 문자열이 주어집니다.  
각 결과는 `O`, `△`, `X` 중 하나입니다.

점수 규칙은 다음과 같습니다.

- `O` : 3점
- `△` : 2점
- `X` : 1점

당신은 각 사람의 결과를 점수로 환산해 총점을 계산한 뒤, **점수가 높은 순서대로 이름을 담은 문자열 배열**을 반환하려 합니다.  
점수가 같다면 **이름을 사전순(오름차순)** 으로 정렬합니다.

# 입력

2차원 문자열 배열 `arr`가 매개변수로 주어집니다.

* `1 ≤ arr의 길이 ≤ 1000`
* `2 ≤ arr[i]의 길이 ≤ 31`
* `arr[i][0]`은 길이 1 이상 20 이하의 알파벳 소문자 문자열(이름)입니다.
* `arr[i][1]`부터는 `O`, `△`, `X` 중 하나가 주어지며, 모든 i에 대해 그 개수는 동일합니다.

# 출력

조건에 맞게 정렬된 **이름들로만 이루어진 문자열 배열**을 return 해 주세요.
